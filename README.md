# Chủ nghĩa xã hội khoa học

Website thuyết trình học thuật được xây dựng bằng `React + Vite`, gồm:
- `Trang chủ`: nội dung trình bày về dân chủ xã hội chủ nghĩa và nhà nước xã hội chủ nghĩa
- `Trò chơi`: minigame ô số bí ẩn kèm câu hỏi trắc nghiệm

## Chạy project

```bash
npm install
npm run dev
```

## Build project

```bash
npm run build
```

## Cấu trúc chính

- `src/pages/HomePage.jsx`: trang nội dung thuyết trình
- `src/pages/GamePage.jsx`: trang trò chơi
- `src/components/Navbar.jsx`: thanh điều hướng dùng chung
- `src/styles.css`: toàn bộ style chính
